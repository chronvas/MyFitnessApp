# An compose playground app.
The user can search for a muscle, and a paginated list of exercises for that muscle is returned.

### API:
```https://wger.de/api/v2/```

### Libraries used:
- Ktor HTTP client
- Coroutines
- Hilt
- Kotlinx deserializer
- Paging 3
- Jetpack Compose UI
- Material 3
- Allopen kotlin plugin
- Mockito
- Cash app turbine
